# 🍔 Hungry

Hungry is a modern food delivery application built with Flutter. It allows users to browse meals, search for food, manage their cart, place orders, and pay using either online payment or cash on delivery.

The app follows a clean architecture using the BLoC pattern and integrates with REST APIs for dynamic data management.

## ✨ Features

- 🔐 User Authentication
- 🍽 Browse Meals
- 🔎 Search Meals
- 🛒 Shopping Cart
- ❤️ Favorites
- 📦 Order Management
- 💳 Online & Cash Payment
- 🌐 REST API Integration
- 📱 Responsive UI

## 🛠 Tech Stack

- Flutter
- Dart
- Flutter BLoC
- REST API
- Dio
- Shared Preferences
